What is a div element?

Bucket that groups sections of a webpage together.

What is one way you can identify or "target" a div element

You can identify/target div element using classes. When you are calling a class in CSS, you will target every div with the same class.

Why would you use a div element?

to organize

What is a border property

Allows you to style the border around an element.

What are some of the different border properties

Borders, Border width, Border color, Border sides, Rounded borders

What is the difference between margins and padding?

Margin is between elements and padding is space inside the element.
